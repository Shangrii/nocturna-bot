|----------|-------------|
| server-rendered template → browser | Reminder names/messages from staff are injected into HTML; Jinja2 auto-escaping is the XSS control. |
| Alpine component → reminders routes | The client echoes back the row's `version`; it must never be treated as authoritative server-side (Plan 04 guards this). |

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-06-17 | Injection (XSS) | reminder name/message rendered in the table/modal | mitigate | Jinja2 auto-escaping (default on); `x-text` (not `x-html`) for Alpine-rendered strings; message body only ever in a plain `<textarea>`, never `x-html` |
